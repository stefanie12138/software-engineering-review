二、简答题高频例题

1. 什么是软件危机？

  • 考点：SE2 什么是软件工程
  • 答题框架：定义 → 主要表现 → 产生原因
  • 详细解析：01-软件工程基础与过程模型

2. 简述软件生命周期各阶段任务。

  • 考点：SE2/SE3 软件过程模型
  • 答题框架：需求分析 → 概要设计 → 详细设计 → 编码实现 → 集成测试 → 确认测试，每阶段一句话说明核心任务与产出
  • 详细解析：00-全书重点速查

3. 软件工程三要素是什么？它们之间是什么关系？

  • 考点：SE2
  • 答题框架：过程、方法、工具；过程定义活动顺序，方法提供技术手段，工具支持实现
  • 详细解析：00-全书重点速查

4. 简述五种基本软件过程模型及其适用场景。

  • 考点：SE3.1
  • 答题框架：瀑布、增量、迭代、原型、螺旋；从指导思想、特点、适用场景三个维度列表对比
  • 详细解析：00-全书重点速查

5. 什么是需求工程？需求工程包括哪些活动？

  • 考点：SE4
  • 答题框架：需求工程定义 → 获取 → 分析 → 文档化 → 确认验证 → 管理
  • 详细解析：02-需求工程

6. 需求获取有哪些常用方法？

  • 考点：SE5
  • 答题框架：访谈、问卷、会议、现场观摩、业务资料分析、原型、头脑风暴等；简述 2~3 种方法的特点
  • 详细解析：02-需求工程

7. 简述顺序图与通信图（协作图）的区别。

  • 考点：SE6 UML 交互图
  • 答题框架：同属 UML 交互图、语义等价、可相互转换；顺序图强调时间顺序，通信图强调对象结构与合作关系
  • 详细解析：00-全书重点速查

8. 什么是模块独立性？如何度量？

  • 考点：SE7 软件设计基础
  • 答题框架：内聚度（越高越好）+ 耦合度（越低越好）；设计原则为高内聚、低耦合
  • 详细解析：00-全书重点速查

9. 黑盒测试与白盒测试有什么区别？

  • 考点：SE13 软件测试
  • 答题框架：从依据、是否需了解代码、测试对象、典型方法、典型阶段等维度列表对比
  • 详细解析：04-软件测试与维护

10. 简述软件测试的层次。

  • 考点：SE13
  • 答题框架：单元测试 → 集成测试 → 确认测试 → 系统测试 → 回归测试；每阶段说明测试对象和方法
  • 详细解析：04-软件测试与维护

三、设计题综合例题

设计题共 60 分，是考试核心。以下按图类型整理常考例题与解题入口。

3.1 用例图

例题来源：

  • 图书馆管理系统（读者、管理员、include/extend）
  • 医院预约挂号系统（患者、医生、定时任务）

解题步骤：

  1. 找 Actor
  2. 找 Use Case（动宾结构）
  3. 画系统边界
  4. 建立关联 / <<include>> / <<extend>>

详细解析：

  • 06-软件工程画图教程：用例图
  • 06-综合练习 1：医院预约挂号系统

3.2 顺序图 / 通信图

例题来源：

  • ATM 取款流程

解题步骤：

  1. 确定参与对象（Actor → 边界类 → 控制类 → 实体类 → 外部接口）
  2. 画生命线
  3. 按时间顺序画消息（同步/异步/返回）
  4. 标激活条、alt / opt / loop 片段

详细解析：

  • 06-软件工程画图教程：顺序图

详细解析：

  • 06-软件工程画图教程：类图

3.6 数据流图（DFD）

例题来源：

  • 在线订餐系统（顶层 → 0 层 → 1 层）
  • 快递包裹查询系统
  • 发帖子系统（来自 03 真题）
  • CCMS 信用卡管理系统（来自 03 真题）

解题步骤：

  1. 画顶层图（上下文图）：1 个加工 0 + 外部实体
  2. 画 0 层图：分解为 1、2、3… 加工 + 数据存储
  3. 画 1 层图：细化 0 层加工为 1.1、1.2…
  4. 检查平衡原则

详细解析：

  • 06-软件工程画图教程：数据流图
  • 03-相关真题与解析：发帖子系统 / CCMS

3.7 DFD → 软件结构图

例题来源：

  • 发帖子系统
  • CCMS 信用卡管理系统
  • 在线订餐系统（变换型 / 事务型两种角度）

解题步骤：

  1. 判断 DFD 类型：变换型 or 事务型
  2. 变换型：确定输入流 / 变换中心 / 输出流 → 一级分解 → 二级分解
  3. 事务型：确定事务中心 / 接收路径 / 动作路径 → 一级分解 → 处理每条路径
  4. 优化模块结构

详细解析：

  • 06-软件工程画图教程：DFD → 软件结构图
  • 03-相关真题与解析
